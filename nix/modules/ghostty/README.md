# Ghostty

モダンなGPUアクセラレーションターミナルエミュレーター設定

## ファイル

- `default.nix`: Nixモジュール設定
- `config`: Ghostty設定ファイル

## 設定の要点

- **テーマ**: Catppuccin Mocha
- **フォント**: Monaspace Neon（サイズ17）
- **ウィンドウ**: 透明タイトルバー、バランスの取れたパディング（10px）
- **カーソル**: ブロックスタイル、80%不透明度、点滅なし
- **機能**: シェル統合（no-cursorモード）、タイピング時にマウスカーソルを非表示

## カスタマイズ

`config`ファイルを直接編集してください。変更はターミナル再起動後に反映されます。

## プラットフォームサポート

macOS固有の設定（透明タイトルバー）を含みます。
