# fzf

コマンドライン用ファジーファインダー

## ファイル

- `default.nix`: Nixモジュール設定

## 機能

- **Zsh統合**: 有効
- **デフォルトオプション**:
  - 高さ: 40%
  - リバース表示
  - ボーダー表示
- **ファイル検索**: `fd`を使用

## キーバインディング

- **Ctrl+R**: コマンド履歴検索
- **Ctrl+T**: ファイル検索
- **Ctrl+]**: ghq統合（リポジトリジャンプ）

## カスタム機能

### ghq統合

`Ctrl+]`でghq管理下のリポジトリを検索して移動:

```zsh
# Ctrl+]を押す → リポジトリを選択 → 自動的にcd
```

### パス補完

Tabキーでのパス補完に`ag`（silver-searcher）を使用。

## カスタマイズ

`default.nix`の`programs.fzf.defaultOptions`を編集して動作を変更できます。
